# Component study

This folder contains the codes used for the component study, with both thermal camera codes and plotting codes.
The structure of the component study codes are as follows:

1. Temp_Plot.py plots the temperatures measured from the thermal cameras.
2. Thermal_Cam_Single_AllPixels_Orion.ipynb is the thermal capturing code used for the Orion A150 printer.
3. Thermal_Cam_Single_AllPixels_Creality.ipynb is the thermal capturing code used for the Creality K2 Plus printer.
4. interactive_plotter.py is the plotter code used to extract the \gls{pdt} temperatures.
5. load_displacement_plotter.py is the code used to plot the load-displacement data.
