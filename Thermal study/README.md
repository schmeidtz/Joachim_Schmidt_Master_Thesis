# Thermal study

This folder contains the codes used for the thermal study, with both thermal camera codes and plotting codes.
The structure of the thermal study codes are as follows:
1. Data_Analysis_Thermal.ipynb runs a code which loads all the datasets, aligns for strain and generates plots.
2. Thermal_Camera_Dual_Expansion.ipynb is the code with the capturing GUI for the thermal cameras.
3. preamble_data.xlsx is the Excel-file used in the data processing code to align the starting point and fetching all the datasets.
4. The folder "Plotters" hold several smaller scripts generating the final plots used in the study through run_all_figures.py.
